Z0DIAC

Zany Operating Distributions In A Container

About Z0DIAC

Z0DIAC is an upcoming multi-distro bootloader designed for research, testing, and safe Linux experimentation on external media.

It will provide a clean boot chain, isolated environments, and the ability to test multiple operating systems without permanently modifying your device.

Planned Features

  • Multi-distro support: Debian, Ubuntu, Arch, Alpine, and more
  • Safe external boot with no firmware flashing
  • Clean root filesystem pivot into systemd/OpenRC or other init systems
  • Research-friendly boot environment
  • Recovery and testing workflows for Linux developers

Inspiration

Z0DIAC is inspired by:

  • sh1mmer - for Chromebook RMA and shim exploration
  • shimb00t - since it's just like this project, with the conncept of booting OS/Linux dists on chromebooks

How it will work:

zodiac will be set up like SHIMBOOT, and will basically act the same, but more modern and with more choices.

Notes

Z0DIAC is not yet released. I'm guessing it will be released in the next couple months.

It is designed to be a great tool for developers, and for people who are just curious about this!
